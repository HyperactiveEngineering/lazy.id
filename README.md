# @lazy/id

## Quick Start

### Install with NPM

```shell
npm i @lazy/id
```

### Install with pnpm

```shell
pnpm add @lazy/id
```

### Install with Yarn

```shell
yarn add @lazy/id
```

### Generate an Id

```typescript
import { id } from '@lazy/id'

const userId = id('user')

console.log(userId) // AZGeDUu7c0iimkTEOtnSYQ.user
```
